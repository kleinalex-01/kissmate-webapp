# Kiss Máté E.V. Webapp

 Fiatal, feltörekvő egyéni vállalkozó masszázsra és személyi edzésre szakosodva. A kliensnek nincsenek konkrét tervei, tartalmon és színvilágon kívül szabad kezet adott.

## Projekt Célja

  Kiss Máté egyéni vállalkozó személyes hirdetése online felületen, aktuális árlistájának megjelenítése, időpontfoglalások kezelése.
